# 二、智能的定义

这一节的核心，是把智能从“突然冒出来的特殊属性”改写成“反应性连续谱上的认识论切割”。石头受热会膨胀，细菌遇到化学梯度会移动，动物会学习，人会抽象思考。它们之间当然有巨大差异，但差异首先是复杂度与层次上的，不必急着说中间有一道绝对鸿沟。

人之所以爱把智能看成骤然出现，是因为总站在自己的位置设阈值。低于阈值的反应性，被我们当作背景噪音；高于阈值的，才被封为“智能”。这个过程和把连续变化切成一个个事件很像，都是观察者在连续体上动刀。于是智能并不是一个世界底层自带的牌子，而是观察者依照自身参照做出的归类。

这套说法的一个直接好处，是许多老问题会自动松动。人工智能有没有智能，不能先问它像不像人，而该问它在反应性谱上落在哪里；石头有没有“感觉”，也不再是非黑即白的追问，而变成颗粒度问题。你因此把争论从“本质上是不是”引到“在何种认识论切割下被识别为是”，路子就开了。
