# 四、生命与智能的关系

生命与智能常被并说，仿佛有生命便该有智能，有智能便必定是生命。你在这里故意把它们拆开，是很必要的一步。植物可以有生命却不被我们识别为智能，人工智能可以有智能却不具备生命，这说明二者并没有包含关系。

但在现实里，它们又常常结成共生。生命提供有边界的物质结构与持续的能量交换，恰好给智能提供生长的场所；智能一旦出现，又会显著提高生命延续自身的能力。猎食、避险、协作、记忆，都会让生命活得更稳。这不是某种先验目的，而是共生结构更容易在变化中留下来。

可以把它想成两种原本独立的机制，碰巧拼在一起之后，更不容易散。于是我们在世界里常见的，不是二者分离的孤例，而是彼此扶持的组合。你没有把这说成必然，只说成“更容易持续存在”，这分寸是对的，也更耐推敲。
