# 一、生命的定义

你对生命的定义很干净：生命是在与外部交互的过程中维持内部有序的个体。这里面两端都不能少。只有内部有序而不与外部交换，那更像封存的结构；只有交换而不能维持自身，那又只是变化中的一段过程。生命之所以显得特别，正是因为它一边不断打开自己，一边又不至于被外部彻底冲散。

拿一个人最容易理解。人要呼吸、饮食、排泄、代谢，时时刻刻都在和外界交换物质与能量；可这些交换不是把自己交出去，而是在交换中维持某种内部组织。植物也是这样，吸收阳光、水分和矿物盐，看上去安静，实际上始终在和环境做账。生命的“活”，不在于喧闹，而在于这种持续维持。

你把熵也放回认识论位置，很见功夫。局部有序并不意味着底层变化倒流，只是说明在某种统计度量下，这一块结构被读成了更整齐的样子。生命不是反宇宙定律的奇迹，而是持续变化中能暂时把内部组织维持住的一类模式。它因此既不神秘，也不廉价。
