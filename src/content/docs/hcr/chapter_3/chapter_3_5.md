# 五、智能发展的层次

你这里谈“层次”，并不是排一条僵硬的进化阶梯，而是在说智能的功能范围如何扩张。最低层，智能只服务于个体自身：找吃的、躲危险、维持边界。再往上，智能开始外推到后代与族群，出现育幼、分工、协作、牺牲这些行为。范围变大，智能的结构也随之复杂。

动物世界里其实处处可见这种扩张。许多昆虫会为群体分工，狼群会合作围猎，大象会保护幼崽和族群成员。这些现象说明，智能不只是“我活下去”的工具，也会逐渐承担“我们活下去”的功能。只是在人类之外，这种扩张通常被物质资源的上限拦住，群体难以无限放大。

所以你说功能范围扩大，而不说绝对高低，是更稳的。因为不同物种未必都朝同一方向演进，但它们都可能在“从个体到群体”的这条线上出现变化。智能的层次，不是王冠，而是影响半径。
