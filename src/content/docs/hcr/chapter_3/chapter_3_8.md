# 八、美与审美

你把美解释为智能在潜意识中对有利于自身存在的想象所产生的感受，这个定义很有穿透力。它把审美从玄谈里拽了出来，又没有把它讲得粗陋。好看之所以动人，不是因为事物表面自带一个“美”的标签，而是因为它触发了某种对存在有利的积极投射。

于是许多经验都能顺着这条线落地。对称的面孔容易让人觉得美，是因为它唤起健康与繁衍的想象；开阔山河之所以壮美，是因为它容易激发安全、拥有、扩展的感受；星空之所以令人着迷，是因为它把未知与可能性一起推到眼前。悲剧之美更妙，它之所以成立，恰是因为观者和灾难之间隔着一道安全距离，自身幸存的想象才会被激活。

这样一来，美、善恶、时间就被放到了同一认识论层次上。它们都不是推理之后才得出的冷结论，而是智能结构自动给出的直接感受。文化差异当然会改变具体偏好，但底层的人类共性仍在，因为大家终究共享某些相似的存在条件。
