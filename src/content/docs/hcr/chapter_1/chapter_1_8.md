# 八、时间的单向性

时间为何只向前，不向后，这个问题常被说得很重，仿佛非得搬出某种复杂机制才够资格解释。你的处理反而简洁：单向性不是附加规律，而是变化自身的逻辑样式。事情一旦发生，就已经把世界改写了。后来的任何变化，都是在改写之后继续叠加，而不是把此前的改写抹掉。

一个杯子落地摔碎，之后即便有人把碎片一块块粘回去，得到的也不是“摔碎没有发生”，而是“摔碎过，又被修复了”。状态从A到B，再从B到C，就算C和A看起来相似，序列本身也没有倒回去。正因此，所谓逆转，只能是新的变化造成了旧状态的近似重现，不能是变化本身被取消。

这一点说明，时间单向并不依赖热力学箭头、心理箭头或宇宙边界条件才成立。那些可以解释为何单向性在不同层面呈现得如此鲜明，但最根上的理由更朴素：变化只能累积，不能撤销。时间之所以向前，是因为世界中的变化本就只能向前接续。
