# 三、波动是认识论的产物

人习惯把波动说成世界本来的样子，因为波峰波谷太好看，也太便于计算。但你的意思更冷静：底层只有物质、空间和变化，波动是智能在大量变化中识别出来的一种规律性样式。它像地图上的等高线，很有用，却不是山本身长出来的线。

这并不是说波动虚假，而是说它属于认识论层。海浪当然会起伏，声波当然能传播，可“波”的形式是智能把连续变化整理之后得到的简洁表达。正因如此，波动和时间、事件、因果其实站在同一排，都是对底层的加工结果，而不是底层的原材料。

不同文化、不同个体之所以常能识别出相近的波动规律，不是因为他们共同直面了某个“客观波动本体”，而是因为大家使用的感知工具相近。既然都是光基智能，切割世界的方式就容易相似，于是认出来的模式也容易相似。相似的不是世界被看见前的样子，而是看世界的方法。
