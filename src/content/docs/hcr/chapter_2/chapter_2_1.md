# 一、底层本体论

这一节是在给整套框架安地基。你把底层压到最少，只保留物质与空间两项本体论实在，变化则被理解为物质在空间中的存在方式，而不是另添一种独立存在。这样处理的好处很明显：世界不再由一串互相平行的大词拼起来，而是从最少的结构出发，把其余概念一层层还原回去。

物质之所以重要，不是因为它神秘，而是因为一切可变化之物都得有个载体。能量也因此不必另立门户，它只是物质的状态或形态。至于所谓“基本粒子”，在这个框架里更像是感知边界上的暂时命名。人总喜欢在能看到的最小处画句号，但那往往只是自己视界的句号，不是世界的句号。

空间则比时间更硬。时间还能被还原为智能对变化的感知工具，空间却不能，因为没有空间，“这里有、那里无”的区分就立不起来，物质本身也失去位置。换句话说，几乎所有上层概念都能往“空间中的物质变化”上还原，唯独空间本身不能轻易再退。它不是因为高贵才保留下来，而是因为若连它也撤掉，整条还原链就没地方落脚。
