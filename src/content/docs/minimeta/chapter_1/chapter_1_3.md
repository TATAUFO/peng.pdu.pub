# 三、极简宇宙模型

要想知道时间是不是底层实在，最稳妥的办法不是一上来就往宇宙里添概念，而是反过来，把概念一层层拿掉，看最后还剩什么。于是这里先留下最小的描述：有物，有运动。除此之外，不预设时间，不预设距离，不预设速度，也不预设质量。这样做的好处，是所有后来出现的概念都必须自己证明自己，而不能借旧习惯偷渡进来。

“盒中积木”的设想，正是为了让这个模型可见。一个黑暗封闭的盒子里，积木漂浮、碰撞，没有钟表，没有太阳东升西落，没有任何现成刻度。若你就在盒中，不是全知的旁观者，而只是其中一块会受撞击的积木，那么你对外界的一切了解，都只能来自别的东西对你的影响。这个设定很朴素，却一下子把问题逼到了根上：所谓世界，对观察者而言，首先不是“客观全景”，而是“我接收到的变化”。

当高速的小积木不断在大积木和观察者之间往返，感知就出现了。观察者从大量连续、带方向的撞击里，逐渐重建出别的积木的位置、运动和变化趋势。时间也就在这里冒头了。它不是预先摆在盒子里的背景，而是观察者为了整理这些持续到来的变化，为了给一切运动找一把共享尺度，才形成的工具。换句话说，时间不是盒子的组成部分，而是盒中智能理解盒子的办法。
