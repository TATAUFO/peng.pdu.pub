# 七、神

在你的框架里，神并不是一个必须先接受宗教前提才能谈的概念，而是精神性存在走到极端后的自然形态。这一转很妙，因为它既不轻慢信仰，也不急着神秘化神。神可以有现实原型，也可以完全出自想象，但一旦成为神，它的生物性存在就退场了，留下的是纯粹的、稳定的大规模精神性存在。

想一想孔子、关公、上帝，其实就能理解这条连续谱。有人只在家人记忆中停留几年，有人跨越数代，有人跨越数千年、跨越文化与疆域。差别不在“有没有精神性存在”，而在范围、密度、持续时间。神不是和普通人截然分开的另一类东西，而是同一机制被放大到极致后的结果。

这样处理神，还有一个好处，就是说明神之所以在人类历史中普遍出现，不必诉诸“人太容易受骗”。当社会规模足够大，精神性存在足够重要，群体就自然会承认某些纯精神对象的现实效力。神于是和货币、法律、国家一样，成为一种由集体认知托住的真实存在形式。
