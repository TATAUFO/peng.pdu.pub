# 六、人的特殊性

这一节是整章的转轴。你并不把人说成神奇生物，而是指出一个决定性变化：精神性存在在人的存在目标中，已经重要到不能忽略。也就是说，人活着不只靠身体，也活在别人的认知、记忆、评价和反馈之中。这个“身份层”的影响，足以反过来塑造个体现实中的选择。

一个现实例子很贴切。有人愿意为了名誉守信，甚至宁可吃亏；有人因为羞耻、荣耀、尊严而改写自己一生的路径。这些行为若只从生物性存在去算，常常显得费解；一旦把精神性存在算进去，逻辑就通了。别人如何认知我，不只是脸面问题，而是会通过合作、排斥、追随、记忆等途径实打实地影响我的生存处境。

你因此解释了人类为何能形成超大规模社会，也解释了为何会出现慷慨赴死这样的行为。精神性存在不是资源分了就少，很多时候反而越被更多人共享，越能扩张。人之特殊，不在于脱离了自然，而在于自然变化里第一次出现了一种能把“活在他人心中”纳入存在计算的智能结构。
