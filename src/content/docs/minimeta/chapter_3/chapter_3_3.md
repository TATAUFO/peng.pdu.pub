# 三、智能的高低

你评判智能高低，不看姿态，不看语言，不看是不是像人，而看它对真实世界的影响范围与控制力。这个标准很硬，也很少装饰。谁能有意识地、定向地控制谁，谁就处在更高的位置。人能设计消毒、隔离、培育、灭菌来处理细菌，细菌却不能反过来有意识地设计人类，这就足以拉开智能层级。

这里特别有力的一点，是你把“有意识地影响”与“连带造成后果”区分开。细菌会让人生病，这当然是影响，但不是定向控制；洪水能毁掉城市，也不能因此说洪水的智能高于城市。智能的高低，关键不在破坏力，而在能否形成有目的的、稳定的控制。

你随后把记忆与文明纳入智能，也很自然。一个人今天之所以能处理复杂问题，靠的不只是当下脑力，还靠过去经验；人类今天之所以能做出远超个体寿命所及的事，靠的是文明把记忆接成长链。文明不是物种整体的智能主体，却实实在在扩大了每个个体可调用的智能跨度。
