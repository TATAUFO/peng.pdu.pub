# 208. 统一思想与群体消亡

在一个社交应用中，更多的真实用户总是好的，并没有数量上的限制。但更多的虚假用户，虽然依旧是用户，但却是不好的，会影响到其中真实用户的使用体验。如何解释这个问题？

可以将上述问题等同于如下问题：为何在一个群体中，增加真实的个体数量会有利于个体存在，而增加大量行为一致的个体却不利于个体的存在。

好和坏的唯一评判标准或者说其跟本的含义就是是否有利于群体的存在。在一个由真实个体构成的群体当中，所有个体对于好坏的概念判断都是独立的，但其标准是在一个统一的标准周围呈正态分布。在这种情况下，每个真实个体对于整个社会所执行的行为和收到的反馈是基本一致的，或者说是平衡的。这种平衡的结果就是个体的行为及反馈不会对整个群体的是非观产生脱离客观自然变化的影响。

但当个体被大量行为一致个体所包裹时，单独个体即便依然按照整个群体的是非观而产生行为，但收到的反馈却与行为并不匹配，而这种不匹配如果持续发生，且指向某个方向，则这种反馈会改变被包裹的个体的行为逻辑，使之偏离整个群体的是非逻辑。其结果就是受影响的个体行为会向并非群体最优的方向发展，而是向局部最优的方向发展。而且这种局部最优也并非真实的局部最优，由于统一行为个体占据比例较大，所以这个局部的最优将只是那些统一行为个体最优的方向。

制造统一行为个体在社交应用中常用的方法就是注册大量用户身份，在真实世界中就是通过不鼓励独立思考的教育，或者高强度的训练。

其实由统一行动个体组成的群体能够作为整体，提高执行效率，比如纪律部队。但其只能服从于上述对错的统计期望值的时候才适宜存在。这种个体尤其不适宜广泛的存在于正常的社会结构当中，其超过数量的存在会同质化原本独立的个体，最终将群体引导到不利于群体持续存在的路径。

即便对于一个理想化的，由理智个体构成的群体，大量统一行为个体的加入，也会为群体带来消亡。
