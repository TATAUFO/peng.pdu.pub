# 185. 忒修斯之船

关于忒修斯之船，问题本身已经错了。

不应该追问换了船板的船，哪艘是忒修斯之船，是修好的那艘？还是用拆船木料拼凑的那艘？

需要理解忒修斯之船从一开始就不是指的那些木头，甚至这个名字也不重要。从开始建造这艘船开始，一个个木板的拼凑，一枚枚铁钉的砸入，都是成就这艘船作为众人眼中单独个体的过程。某个时刻，这个个体被赋予了忒休斯的名字，这个赋名时刻同给船再凿入一个铁钉，或多拼接一块木板没有本质的区别，不过是给这个被视为船的个体再增加一个事件的累积。所以当船出航之后，无论换下多少块木板，这艘船依然是原来的个体，而那些由拆船板建造的新船，则是另一个新的个体，当然名字是可以随意起的，甚至还叫忒修斯，但这并不是那艘船，原来的忒修斯之船还在。

> 亚里士多德质料因变了但形式因没变的解释是因为那时依然将物看成是实体，不知是不是到维特根斯坦才变.

> 其实个体的定义来源于智能对于事物的认知，所以有人认为拆船板组合成的新船就是忒休斯之船又何尝不可呢。我们将桌子上的电脑，鼠标，水杯看成三个东西，同时却将满是灰尘微粒的显示器看成一个东西，而下意识不去计算由于灰尘微粒造成的个体数量的增加。个体终究是个主观的定义。
