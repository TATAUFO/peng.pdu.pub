# 175. 定义信仰

何为信仰？

事物都是通过有利于个体或群体的存在来体现价值。有些是直接的，比如食物；有些是间接的，比如一座桥。日常生活中间接的事物占据绝大多数，其价值都需要通过转换或交换来体现。而信仰则属于精神范畴内直接有利于精神存在的事物，其不需要交换即能够有利于个体的精神存在，体现价值，即精神食粮。

> 直接的事物也可能属于间接，比如一块面包，肚子饿了自己需要吃就是直接的，而面包房里售卖的则是间接的。信仰也如此，如果相信上帝或佛祖，相信神能够在苦难的时候带来心灵的慰藉则是直接的，如果借由其来获取权利或者信众的服从则是间接的。在信仰的群体中必须有个体能够直接从信仰本身获取价值，否则其与外界的利益交互无从开始。信仰只能从理解和相信开始，而不能从价值交换开始。
