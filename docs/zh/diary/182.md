# 182. 波粒二象性及尺寸维度

微观粒子具有波粒二象性的另一种解释是因为此时的粒子体积已经到了尺寸维度的边缘，更大的体积就是属于当前观察着所存在的尺寸维度，而更小的尺寸则属于另一个尺寸维度，那个尺寸维度对于观察者来说是不精确可知的，甚至是不可知的。

尺寸维度并非一个绝对的概念，而是对于应观察者所代表的智能而言。观察者所能比较精确观察的事物，都属于这个观察者背后的智能所存在的尺寸维度，更小的尺寸对应观察者而言是不能独立感知的，只会受到其统计意义的影响，也是不确定性的来源。更大的尺寸维度带给观察者的影响也可忽略，因为其对于观察者的时间来说，几乎可以认为是恒定不变的存在。

> 尺寸维度跟智能所感受的时间是相关的，智能的升级应伴随有尺寸维度的拓展。
