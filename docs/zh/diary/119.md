# 119. 家庭

家庭做为最基础的社会组织形式，其内部不存在类似货币的信用机制，取而代之的是天生亲缘关系构成的相对固定结构及信用机制。

其结果是所有工作都由高效的个体负责，而低效个体则不负责任何任务，这个机制能给予类似的个体以发展的机会。在时间维度上平衡了单独个体的存在的回报和付出。而整体社会则可以看成是空间维度的平衡，即同一时间点的个体间平衡。

家庭中自然存在的亲缘关系的天性，取代了货币为基础的信用计量。

「在家庭的单位中，亲缘关系的存在造成个体间如果抛开时间维度，在个体差异上其实比较小，儿子的智力体力跟父亲会很相似。这在某种程度上减弱了对于货币这种的统一工作量证明的需求。」
