# 88. 意识和智能

生命并非都有意识，也并非都有智能。

智能的下限低于意识，智能是意识的基础但不一定能升级为意识。而生命也不一定有智能，低级的智能服务于生命个体，而当存在意识，则生命个体服务于个体意识。

区分是否有意识的表象在于是否有对于权力的争夺，比如蚁群没有，天然的。「这里说的不一定对」而狮群有，有意识的。

> 智能的概念，即一种能量在有限的物质范围内不断运动，且改变物质形态的过程。
