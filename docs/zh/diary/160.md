# 160. 好的群体和坏的群体

好的定义是利于群体，「与是否利于本个体无关」，而坏的定义是不利于群体存在「其实也与是否有利于个体无关，只是即不利于群体也不利于个体的事，缺乏发生的动机」。或者说好的定义为将群体的存在优先于个体，而坏是将个体存在优先于群体存在。

好的个体在坏的群体「由主要实行坏行为个体组成的群体」中持续好的行为受到的惩罚远大于坏的个体在好的群体中持续坏的行为受到的惩罚。因为惩罚本身就是对个体而言，但好的个体在坏的群体中做好事，并没有足够的利于个体的利益去平衡。

囚徒困境无法改变，只能离开而进入到好的群体或更坏。好的个体在坏的群体中受到的惩罚不是一次性的，而是持续的，结果就是持续性的无所得。
