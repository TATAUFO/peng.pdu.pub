# 235. 战争不会消亡

> 人类是唯一从事那种骇人听闻之暴行——战争——的动物。他们会聚集同胞，冷血且镇静地出发去消灭自己的同类；人类也是唯一会为了一点卑微的报酬而去成批屠杀那些与自己素不相识、无冤无仇之同类的动物……（节选自马克吐温的The Lowest Animal）

区分于所有其他动物，人类群体性暴力的主要对象恰恰是同类。这根源于人与动物的根本差异：人类的精神性存在与生物性存在同等重要，而动物并不具备这样的“精神性存在”。在精神层面上，一方面，越庞大的同类族群其中每个个体便能获得更长久、范围更广阔的精神性延续，这也是人类族群不断扩张、合并的原因；另一方面，精神性需求与物质需求并不相同，它所争夺的并非物质资源，而是精神层面的排序。在一个固定的群体范围内，精神排序通常是零和博弈，这便是战争难以避免的根本原因。

当外部资源充足时，虽然也会因争夺资源而产生冲突，但往往只是小范围、短期的。一旦外部资源无法继续拓展，就会引发更大规模且更持久的冲突。人们常将其理解为对资源的争夺，实际并非如此：当可拓展资源存在时，人们对物质的满足在一定程度上削弱了对精神满足的追求；可当资源无法拓展、物质需求难以继续满足时，精神性的需求就会凸显，从而引发更深层次的冲突与战争。

基于精神性需求导致的战争，也可以视作自然对人类这一智能物种数量的限制条件。一方面，精神性存在需要更大的族群规模；另一方面，精神性的满足又会使人类出现自相残杀，从而削减群体数量——这似乎形成了一种自然的动态平衡。

宇宙中任何事物都存在制约与平衡，这种平衡是普遍且基础的。这并不意味着平衡永远不会被打破，而是如果它在过去曾被打破，宇宙早已走向毁灭；如果未来某天平衡被打破，宇宙也将在不久后毁灭。
