# 210. 教育的区别

在军队的训练过程中往往要花很长的时间来训练看似毫无意义的队列行进或将被子叠成完美的形状。这些训练内容的目的在于让受训者习惯于服从，不因为自身对事物的判断，而是因为服从而去做事情。

这个训练的过程往往还包含着人为赋予这种无意义的事情一个含义，让被训练者相信。如果成功的话，说明被训练者不但服从，而且容易接受编造的概念。这样在需要受训者做其他的事情时候，受训者也不会有独立思考的能力。

现在教育也在向这个方向发展，大量高压重复的练习，剥夺孩子的一切时间，让他们疲于执行而没有时间思考。这种训练不但成功的影响了学生，也将老师和家长都卷入其中，让每个人都不得已的跟随这个节奏生活，没有独立思考的时间，也没有判断的机会。
