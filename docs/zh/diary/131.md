# 131. 时间停止

我们观察一个以光速运动的物体。物体被观察者认为时间停止和物体本身时间停止是两回事。~~A，B相对光速分离时，A看到B静止，B也看到A静止。~~

「划去的部分想错了，光速的恒定不是对于光源而言的。」

「真实的时间停止，应该是无事件发生的。而被观察的时间静止是没有被可观测的时间发生」

系统并不存在封闭系统，宇宙是唯一一个实体，来自于斯宾诺莎。可理解成只存在一个实体，任何从个体智能去观察的系统都不是封闭系统。
