# 217. 光速旅行

根据人类的感知，光是速度最快的，也只有光才能达到光速。

光是和我们以光速相对运动的其他物质？

相对论告诉我们，当物体高速运动时，其质量增加。根据公式，当速度趋近光速时，物体的质量趋近无限大。这一概念限制了我们无法通过人为加速使两个物体达到相对光速运动。但相对论认为没有特殊的参照系，因此质量变化也是相对的，类似于时间和空间，质量也是相对的而非绝对的。

然而，考虑到光和光速存在，也许相对光速运动不仅仅是质量增加的过程，还可能涉及另一种变化，即物态的相对变化。设想一个物体在加速过程中，可能存在我们目前无法观察到的光化过程。在低速情况下，光化作用微弱，无法抵消速度增长带来的相对质量增加，但随着相对速度增加，光化过程对物质的影响逐渐超过了相对速度带来的质量增加，直至趋近光速时，光化作用占主导，质量反而趋近于零。

以一艘正在加速的飞船为例。在低速阶段，观察者感知到飞船由于推力输出而逐渐加速，相同的推力使得飞船质量增加，导致加速度下降，但这仅限于观察者视角。随着速度进一步提高，飞船的光化过程逐渐可察觉，飞船对观察者不再是完全的实体。光化过程可能在相对速度低于光速的情况下难以观察，但达到光速时将完成。因此，真正找到并观测到这样的物体可能十分困难。

尽管我仍然相信相对论的正确性，但在两个相对光速运行的物体之间，彼此都会认为对方是光，且是巨大的能量。但对于自身观察者而言，他们都会认为自己以实体形式存在。

这种现象无法被观测的原因在于，当一个物体完成了相对于观察者的光化过程后，其对光的反射消失，或者说基于光速的观测系统无法探测到以光速运动的物体。

至于黑洞和背离我们方向的光速运动的物体可能有类似的观测结果。
