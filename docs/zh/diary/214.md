# 214. 庞加莱猜想

庞家莱猜想说的是对于一个封闭的，有限体积的系统，经过足够长的时间，系统的状态将会重复，即系统将回到最初始的状态。

庞家莱猜想和熵增理论虽然有不同的适用范围，但看上去依然是矛盾的。如果熵增是对的，宇宙的终点在于热寂，而如果庞加莱猜想是对的，那宇宙便是循环往复的。熵增可以推论出宇宙是单向的，是一个由宏观向微观尺度塌陷的过程，而庞加莱猜想是一个循环的过程，此过程也可看成是宇宙存在的波动性的体现。
