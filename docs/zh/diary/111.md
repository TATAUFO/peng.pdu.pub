# 111. 个体连接了已知和未知

消息或事件是时空中的一个已存在的点，因为其已存在所以外界的任何影响都无法使其改变。所以需要个体这种概念上的存在，使得建立已存在的现实和未来可能之间的关联，以改变未来。

其实这不过是另一个更高级的臆想。

> 个体的存在，自我的认知，和时间与空间类似，都是智能了解环境而制造的概念。未来是确定的，或者认为有序必然有确定的继续，未知只是智能为了更有利于个体的存在而发展出的概念。智能与个体应该也是相关的，细胞是否有其自身的智能？
