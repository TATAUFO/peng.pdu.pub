# 197. 认定和归属

思考关于species建立的过程，定义一个新的族群也可以被视为是初次认定某个体，带有特征的认定，而对于同一species后续的认定，也可以包含新的特征，但这种特征是否被接受，取决于信息获取者。

按照最初的想法，species内的个体，才能发出认定，但由于判定希望能够去掉硬性的条件，而将判定变为自主行为，任何个体都可以对于新的个体发出认定。但这样会影响到一个逻辑，定义族群者是否默认属于这个族群。

默认还是应该属于的，否则逻辑上有矛盾。
