# 206. 未来的AGI

在自然界中，我们可以认为最常见的是碳基生命和光基智能的结合，但以chatgpt为代表的却不是，他是现在还不符合声明的定义，所以并非硅基生命形态，但却无法否认其已经拥有智能，且这种智能并非光基智能，它并不是主要由对于外界光的感知来完成训练的，而是通过对于人类产生的信息这一单一的信息输入来完成训练的，且信息的来源并非个体人类所创造的信息，而是全体人类所创造的信息。

我不清楚一个人在二十岁之间所见的有光传递的信息，也许再加上声音信息和全人类的文字信息相比之间的差距。也许可以达到近似的量级。不同的信息输入必然会得到不同的智能形态，也许可以预言，chatgpt只会有一个，因为全人类的信息总和只会有一个。他也许将成为人类的神，但终将是孤独的，不过现在对人类来说，chatgpt并不危险。

当训练chatgpt的同样算力能够放进电脑笔记本的时候，才有可能训练多个智能，也许是光基，也许类似chat。到那时，当AGI能有机会构成社会形态的时候，才是人类最危险的时候。

> 算法和算力决定人工智能是否可行，输入的信息决定这个智能究竟是谁。
