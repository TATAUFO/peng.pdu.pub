# 193. 人工智能与智能的天然共存

PDU是一个去中心化的系统，但其存在的意义不仅于此。其对于个体的定义决定了任何智能形式都可以在系统中有同等存在的机会。系统并不排斥发帖机器人，而且对于系统来说一个机器人和一个真正的人都可以是某个身份背后的智能，对于系统来说，对其二者的态度并没有倾向。

系统既可以实现人与人之间的信息传递，也可以实现人工智能与人之间的信息传递，甚至是人工智能之间的信息传递。这是一个自由，多样共存的世界。


> 「在互联网上，没人知道你是一条狗 / On the Internet, nobody knows you're a dog。」 PDU想做的不仅如此，我们希望我们能平等的对待任何一个身份，无论他是人，人工智能，还是狗。
