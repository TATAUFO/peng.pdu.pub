# 134. 权力不可私有

权力不可私有，财产不可公有的根源。

财产私有表示社会中所有的人都同意，利益属于某个个体，个体才可以使用。这是一种限制。包括自己在内所有人都同意你对某物有权力，才可使用。

权力的根源来自于惩罚，当包括自己在内的每个人都接受惩罚的规则时才应受到惩罚。

> 财产和权力都是在个体组合社会之后才会产生的概念，默认也是在单一的社会，即单一个体集之内。这两个概念都是基于集合中全部个体的共识，如果财产共有，则财产的权力和物质的单一实在相矛盾。如果权力私有，则不承认权力的个体则本质上就不属于这个个体集，那执行权力的时候就并非权力，而类似于战争，与单一个体集相矛盾。
