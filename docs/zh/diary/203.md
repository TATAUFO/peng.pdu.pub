# 203. 计划经济，更复杂的市场

经济学有时会将经济的形式简单的分为市场经济和计划经济，或认为计划经济因为破坏了市场，便是一种低效的经济组织形式。我们换个角度看这个问题，计划经济何尝不是一个完全的市场，只不过市场中所所售卖的东西不仅有商品和服务，还有能够间接获得商品和服务的权利，这与期权没有本质的不同，权利的售卖也附带更高的风险和利润。

在这个市场中，所使用的一般等价物也不只有单一货币，还有时间及在交换权利过程中付出的权益，比如要长期的忍受上级权利的压榨或对上级保持违心的逢迎，这些可能都是需要付出的代价，一种精神性的代价。但在社会这个由人的精神存在而构成的结构当中，这些精神上的权益的付出可被视为一种真实存在的，实实在在的付出。

虽然这个市场与纯市场经济没有什么不同，但因为人的存在更注重精神存在的原因，所以当一种经济模式所规定的内涵不只包含纯物质的范围时，精神的市场就必然会挤压物质市场的空间，结果就是强迫所有市场的参与者参与到这个包含精神的市场当中，而对于想单纯的在其中寻找物质市场的个体进行压榨。

但精神的市场与物质市场最本质的区别在于，精神的市场是一个零和游戏，而物质市场不是。物质市场鼓励个体从自然界中获取更多的物质，冲入到市场当中，而精神市场所交易的东西必然是将精神的价值从一个个体转移到另外的个体，有人发号施令，必然有人服从，否则发号施令者得不到任何精神上的好处。不过扩充精神的市场还有一个方式，就是对于大众群体宣扬服从性，比如统一思想，这样可以减少大众在服从过程中的痛苦程度，从而来增加精神市场的总体供给。

这个市场依然有看不见的手在调整供需平衡，那个平衡点也许不是价格，而是公务员的录取比率。
