# 77. 智慧与自我

个体的概念是智慧对于世界中与自我相同概念的演绎。

智慧由对世界的感知而定义了**自我**或**主体**的理念。然后以此为基础在世界中寻找类似于**自我**的客观存在，而产生**个体**的理念。

> 任何一物都是由主体由自我认知而在外界环境中寻找的某种程度的类似，比如猴子是我们作为灵长类动物的类似，石头是作为单一存在之物的类似，单一存在应该是这种类似的抽象概念。
