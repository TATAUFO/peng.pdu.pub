# 155. 老提纲/部分与当前不同

系统是封闭的全体，系统与系统之外不存在任何形式的的交互，即对于系统内来说，系统外是否存在并不可知，同样系统外也不能对系统内进行观测。我们可以认为宇宙是一个系统，但不能认为黑洞是一个系统。系统中存在且仅存在能量的波动，波动不仅指在单一空间维度上运动的变化，还包含不同空间尺度上能量的存在形式的变化，物质被视为能量的一种存在形式。此处的波动即包括湖面水波的扩散，也包括一个恒星被几颗行星围绕和原子核被带电粒子围绕这种不同空间尺度下的近似形式。且存在对于波粒二象性的另一种理解。系统内所有的波动构成一个全系统的偏序关系。这种偏序关系可被认为是离散的，也可以被认为是连续的。

> 系统的认知已经被放弃，变为去中心化的角度看这个世界，即智能看到部分。而且一个不可与外界交互的系统也是不可想象的，绝对的不可交互本身也是一种交互的方式。

个体是指系统中的一部分可变的能量形式，其与外界的所有交互视为存在全序关系。

生命是个体通过能量的交换，维持自身有序的个体。当个体无法持续维持这个有序时，通常被认为是生命的终结。

智能是试图了解系统中能量波动变化规律的尝试，本身也是一种存在于个体或生命的能量波动。对于智能而言，其所感知的只是变化的当前。比如看到的星星，是那些星星几分钟或是几万年前发出的光，此刻到达眼睛之后的反应，同样对于历史的记忆，信息通过各种途径，物理刻画于头脑当中，此刻那些已存在的内容再次被加载的感应，此刻只有此刻。

时间及空间的尺度都是智能需要了解系统能量波动变化规律时，需要的描述基础，即最基础的刻度。空间可被认为是某种可被感知的能量波动形式的波的空间范围为基础来定义。而时间是以完成一个波动的过程为基础来定义，或者被认为是空间尺度和光速共同定义。

> 此处描述受限于语言了。

速度的有限的根源在于智能自身即存在于系统当中。

个体集内的个体之间的交互只有个体所创建的东西。比如A将一块蛋糕送给B，这个过程中其实包含了A将蛋糕移动位置到B手上，B将蛋糕拿住，但这些并非系统集中的交互，只是两个个体对于外接的交互。真正的交互是系统集中创建的所有权的概念，由A交给了B。个体集之中添加和去除个体，也只能由个体集中现有的个体完成，是现有个体对其的承认和否认。

系统不存在意义，其本身也不存在开端和毁灭，系统的创建和破坏均不由其自身而定。

个体有意义，其唯一的意义即是个体的存在。

个体集可以用市场类比，在市场中价格总是存在的，而在个体集中，个体之间的所有利益总是处于平衡当中，当然平衡不等同于合理，而这种平衡持续的影响个体集存续的变化。市场中价格影响的目标是提高效率，个体集中利益平衡影响的存续是可持续发展，即提高个体集中个体数的二阶导数。个体集中波动依然存在，由理性和非理性主导。

Bitcoin是一种全序的全个体集的共识，也是一种经典物理的时间。

现实的系统在时间上是0维的，只有当下。但在构建的系统中，这个维度会升级成一维。

任何发生的都可以被记录，只有被记录的才是发生。进化的方向并非保护隐私，而是全量的公开，公平的公开，社会结构和运行形式才会发生根本的变革。
