# 225. 哲学与科学

哲学是智能对于世界的猜测，而科学是智能对于世界的认知。

科学基于观测，比科学更大的范围就是我们对于世界的观察，而判定科学的标准就是科学要符合这些观察。而哲学本身涵盖的范围其实更大，他包含所谓科学，也包含智能对于外在的观察，哲学甚至尝试去解释这种观察。

> 我能够摸到眼前的桌子就认为这里一定有个桌子，而哲学会讨论有没有可能你的感官欺骗了你，当然你会说这不可能，因为别人也摸到这里有个桌子。但怎么能够确定别人是真实的，像桌子一样是真实的。

> 哲学必然有一个最初的根基，我认为是「存在」
