# 202. 社会的方向

在由下而上的社会结构中，当个体发现问题的时候，个体首先会尝试的解决，解决不了则要求群体解决，此处的群体可以体现为个体选择的某个代表。下层个体能做如此要求，是因为其能直接决定着上一层，即他们的代表。代表的表现不能令这些个体满意时，下层可以方便的对直接上层进行更换。

但如果社会形态变成自上而下，则问题解决会变得困难，如果由于上层造成的问题被下层发现，下层无法解决问题，上层也不会应下层的要求而解决问题，则问题锁住了。更坏的情况是上层还要求下层在不改变上层造成问题的行为这个前提下解决问题。

> 例如学校有大约八百人每日上课，但同一时间下课，且要求通过同一扇大门出校。接孩子的私家车每日放学时都会堵塞交通。如果这个问题由学校解决，可将八百人分散到四个门放学，指定几班到几班从哪个门散学，如果还会造成拥堵，则可以将每个班的放学时间略作分散。但是学校却作为上层却并不想改变自己的行为模式，因而只会发通知给家长，让家长自行解决散学拥堵问题。也许学校也无意解决这个问题，只是做做样子罢了。
