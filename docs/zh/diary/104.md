# 104. 乌托邦只能是表现

佛家讲布施的功德，其也是一种成佛的过程。布施应无分别心，无对于回报的渴望，也应无因由，即布施回报。

共产主义也是在某种程度上是将自己的给予别人，在各展所能各取所需的环境中，相当于给予非特定对象，无分别心的给予。区别在于是整体，还是个体的行为，即便系统中每个个体都采取同样的行为，如果存在强制的规则，也与去中心的个体不同。

> 共产主义社会可能是当社会或文明发展后的表现形式，但不应是一种规则。如果在每个个体都有选择的情况下，依然选择了各展所能，各取所需才接近于理想中的共产主义乌托邦。但如果以此制定规则，则结果可能与意愿是背离的。再举个例子，如果每个人都吃饱了，即便桌子上还剩有食物，大家也不会再吃，这相当于达到了各取所需的物质基础。但不能规定一群饥饿的人必须互相谦让，然后声称达到了某种和谐平稳的状态。
