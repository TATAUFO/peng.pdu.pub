# 200. 质量的想象

如果说世界的本质就如同运动之物模型，则需要解释的概念并不多，比如时间、空间、质量也许还有电荷。用运动来解释这些基本性质，时间便是运动与变化产生的有序带给我们的感受。空和非空是运动的必要条件，无论缺少哪一个，都不会存在变化，即运动。质量则是维持运动不变的原因，也可以看成是能量传递的介质，这些质量在运动状态变化过程中吸收了能量。电荷是什么？也许与更小的尺寸维度中物的运动有关，应该是一种能用其他的基本概念解释的概念。

> 将物质视为一个能量的大尺寸维度，则空间则是对应尺寸维度。

> 如果将质量理解为保持运动状态的原因，需要想清楚的是万有引力定律。质量对于空间造成了怎样的影响，才产生了被感知为万有引力的性质。所产生的力应该不是本质，本质是质量依然要保持自身的运动状态，而万有引力则是对于这种状态维持所产生的表现。当物体相近的时候，引力变大，如果他们的相对运动并未发生变化，那只能理解为他们之间的空间变得更为稀疏，也许在别的空间维度上更加稀疏，想象下好像是被拉大的网格。
