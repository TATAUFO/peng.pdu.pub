# 39. 个体是纯粹理念

个体、个体集、主观世界等只要是系统的子集都可以视为理念，而将能量都归于系统。

> 这里的个体等可视为纯粹理念的存在，而将非理念的一切视为属于系统，即宇宙这个已知且唯一的封闭系统。

- 理念来自于智能
- 智能产生于生命
- 生命来自于个体
- 而个体又由理念定义。

循环么？！

生命和智能可独立。生命是控熵，智能是了解变化。

> 后续对前面四句话有不同的理解，智能与生命是两个独立的概念。生命与个体之间也没有明确的联系。
