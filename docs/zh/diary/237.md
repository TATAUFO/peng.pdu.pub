# 237. 无用之物

黄金和比特币一样都是没有实用性的，为何没有实用性但是有稀缺性的东西会具有价值？这源于人类的社会性，是人类区别与动物的必然结果。社会性的个体所追求的必然包含同类之间的层级关系，即统治的关系，而具有稀缺性的物品则可以有助于体现或者实现这种不平等关系。

如果动物性远大于社会性，比如狼群，则只有具备实际用处的东西才能够提现这种关系，正因为人的社会性对于存在的重要程度能够比肩动物性，所以没有实际用途但具有稀缺属性的物品才会成为个体之间关系体现的媒介。地球上只有人才会使用这种无实际用途的东西来体现个体间关系，而动物之间只会用有实用价值的东西，比如食物，来体现个体间关系。

当然也可以认为这些无用的东西，可以作为有用的东西的映射而存在的。

> 马克思的理论中所强调的“阶级斗争”，或许可以视为对社会关系中“不平等必然存在”的一种理论包装，而且这种包装也许是刻意的。更重要的是，其并非要彻底消解这种不平等，而是试图利用不平等的客观现实，让自身的力量获得持续的统治地位。与此相对，现代民主社会则更倾向于通过共识和规则来抑制或缓和不平等及对立的产生。换言之，这两种思想在承认社会不平等的客观存在后，分别采取了不同的应对策略：前者强调“利用”不平等以获取并巩固主导地位，后者则主张用制度和程序来约束或降低这种不平等带来的冲突。
