# 170. 错误修正

修正错误最简易的方法应是恢复到错误发生之前的状态，而非在这个错误上进行修补。但现实中多数的情况则与之相反，个体总是易于接受错误成为既定事实，然后在其中寻找缝隙生存。这是一个极易被利用的矛盾点，或者说非理智点，像长期利益和短期利益一样。

1. 修补的方式是否能够找到最优解？
2. 个体、群体的关系是否有利于或者不利于求解的过程？

> 由于一切都是有序事件的叠加，所以恢复错误之前的状态与在错误基础上增加一个修正，也无本质区别。
