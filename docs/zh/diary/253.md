# 253. 不存在持续

世界是由事实构成的，由且仅由事实构成。如维特根斯坦在《逻辑哲学论》开篇所说。那么，何为事件？它可以被视为发生改变的过程。此处有一个需要讨论的问题：例如，一个受到重力影响而运动的物体，其速度的改变是否构成一个持续的事件？我认为不是，我认为持续的事件并不存在。事件是一个比时间更为基础的概念。所有看似持续的事件，要么可以被分割成有限的事件组合，要么是一种没有改变的状态。受到重力影响的运动应当被视为后者，我们认为速度的改变是相对某个观察者而言。

与**持续**类似，另外一个不存在的概念是**全部**。
