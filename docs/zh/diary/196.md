# 196. 制服与口罩

现实世界中，容貌的可识别等同于虚拟世界中的信息附加签名。容貌随着年龄的增长会发生变化，但成年之后的这种变化不足以破坏其可被识别的特性，所以通常成年之后的人被作为一个完整的人看待，而幼年时期因为其与成年的容貌有较大的变化，则通常被单独认定，所谓被归类为孩子。

> 容貌的变化具有连续性是事件持续的发生于个体产生的必然结果，但其并不影响个体容貌对于他者的辨识。

单独个体的可识别是社会构成的基础条件之一，可被认为是定义个体的基础条件。但为了某些特别目的，这一基础会被改变。比如军人通常会穿制服，即是尽在军队中，只给军人群体自己看，这是为了让受众，无论是民众还是军人本身都能够明白这些个体在一定程度上剥离了个人意志或特性，而需要作为某个特定的群体性意志的执行者。

演员可以看做是另一个极端，他们极大地突出外表的个性差异，具有极高的个体辨识度，但其形象也并不代表本身的个体意志。但这些通常给受众带来认知的困惑，商业广告中对于明星的使用即是利用这种困惑，让民众将对于角色的喜爱或新任延伸到演员身上。

个体的形象辨识度高且代表个人意志的通常是一些企业或者政治领袖。

若没有强力的规则限制，制服或者面罩、口罩等削弱个体形象的行为将是危险的。这种身份弱化会使社会失序。不仅对于他者，对于穿着制服或者佩戴口罩的人也会有巨大的影响。因为如果缺乏辨识，则个体的行为后果则不需要执行行为的个体单独承担，这里并不仅指犯罪。而当行为缺少反馈的矫正之后，个体的行为极容易向某个方向放大。

类似于制服，一些团体组织也会达到制服的效果。所以团体应该是为了某个目标而存在，而非为了对外部显示为某个同样身份而存在。当某个团体被提及的时候如果总是以团体的标签来标明其中个体的特性，也许是危险的信号。
