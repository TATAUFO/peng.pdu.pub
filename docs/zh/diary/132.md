# 132. 智能与记忆

智能约等于感受「刺激」的反应+记忆。是天然反应在时间维度的升级。

没有记忆，并非刺激和反应会同时发生，而是无记忆的话，只反应一次。或者说反应只由刺激产生而不会因为以前的刺激产生。

智能是物质和能量的结合时由于相互的影响而喊声的一种形态。有电的电脑能成为智能，智能与如何创建是否自身持续都无关。其与生命是不同的概念，即可以有无生命的智能。

智能表现为由持续的能量输入产生系统内部的物质状态高速，微弱的改变，以对外界的影响，只有可反复的物理状态改变。

智能与火不同。
