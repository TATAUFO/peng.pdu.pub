# 186. 社会与个体并存

PDU用信息构建身份，但社交或者发布信息并非目的，在去中心化的环境中创建可使用的身份才是目的。可以将发布的信息看成是信息世界中的行为，而行为的积累才是身份。

电子签名是什么？其等同于我们的躯体，现实中给被称之为灵魂的智能一个全序限制，同时给他人一个可以识别的简易标识。

> 由于有了躯体，所以一个智能不能同时出现在无数个地方，对无数的事件产生影响，所以其相关的事件必然是有全序的。

由信息传递产生的社会形态和个体必然同时存在，相互依存，没有不在社会中的个体，也不存在没有个体的社会。

> 这是元宇宙或web3.0

PDU的想法并非只是描述元宇宙的最终形态，而是给出元宇宙从无到有的路径指引。
