# 220. 中本聪的离开

对于比特币来说最重要的一件事情是什么？

一个答案是中本聪的离开。

我同意。

无论黄金，钻石还是任何货币，其价值都并非来自于其使用价值，而是来自于社会中所有个体的看法。无论什么缘由，当社会中的每个个体都相信其拥有价值的时候，这种本身带有稀缺性的事物才会拥有价值。而中本聪的离开在某种程度上将比特币由一种技术变为信仰，而这种类似信仰的东西加速了参与者的相信。如果中本聪没有离开，没有消失，也许在很多年以后比特币依然会成为一种非常成功的电子黄金，但正由于中本聪的突然消失，极大的加速了这个过程。

所以中本聪的离开也许自始至终就是比特币设计的一部分。
