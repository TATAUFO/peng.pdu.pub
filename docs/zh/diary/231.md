# 231. 群体性思维

很多个体所认为的群体意识是错误的。当某个体对于其他个体利益的考量标准无论高于或低于其他个体，这个思考的个体所认为的群体意识均是扭曲的。如同在经济学中，自由市场的存在，成交价格的存在，其根本的作用都是维护市场参与各方的总体利益最大化，看不见的手是最高效的维护手段。同理，虽然人不是绝对理智的，但如果个体在考虑问题的时候，都以群体利益最大化为标准，那将有助于群体效率的提高。而且当考虑问题时的群体范围越大，对于效率的提高越大。

但由于对于群体效率的提高，和这种提高对于个体的反馈不同，所以在现实中，这种群体的范围划分总是存在不同的范围，而且因人而异。最小的范围不是主体，而仅仅是当下的主体，而最大的范围也不仅仅是已知的所有存在，而是已知的所有永恒的存在。

还有需要注意的是，在现实中，思维的群体变换是随时发生的，正常的，且不是单向扩大的。

> 常见在家庭中，无论对于父母的过度顺从，还是对于子女的溺爱，如果并不是以家庭这个群体的利益作为唯一衡量标准，则均是上面所述的情况。这种情况中，个人往往是按照跟对方的关系来判断各自利益的平衡点，而非以家庭整体作为考量标准。但任何考虑的偏颇，都会损失一定的家庭利益。
