# 218. 面对波动思维一致性

认知波动的关键不止在于相信波动是事物的存在形式，还在于清晰当前所需面对的究竟是哪个波动。

行为方式要符合当前所处理的波动范围及周期，而且对于单一事物而言，应对策略的诸多方面，也应该对应同一个波动。

> 举例来说，对于加密货币市场的参与者而言，首先需要明确自己的交易行为所面对的波动，如果你想面对的是比特币四年为周期的减半行为，则只需要在减半前大约一年的时间，寻找合适的买点，并在减半后的一年半的时间寻找合适的卖点即可，而且对于比特币在这个周期内5至10倍的涨幅而言，购买时30%的差价其实并不重要，卖出是10%或15%的差价也不重要，都应是在可接受的范围之内。但对于一个参与分钟及高频交易的市场参与者而言，同长会用高倍的杠杆在市场中持续的交易，这个时候也许1%的市场突然波动，都会对于参与者带来爆仓的风险。对于长期的以四年为周期的交易者而言，既然你的收益是以四年中5-10倍为标准，那损失同样也应以四年为标准，而非一个星期。这里的含义是，不但对于交易的时间周期要符合所预设需要面对的波动，波动的范围同样要符合所预设面对的波动。
