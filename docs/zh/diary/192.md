# 192. 立地成佛

佛说：放下屠刀，立地成佛

其含义是一种内在的分割个体的方式，也就是分割有序事件的方式。由于某种原因，使得原本具有同标志的有序前后不再具有相关性。所谓皈依往往会附带外在的表现形式，比如改名，剃发，穿上僧袍及吃素等。这些外在的改变是在提醒他者对于变化个体的认知应该有改变。

所以说个体也并非一个严格的概念，单独个体对外必然是有全序的事件，但并非全序且附带标志就是不可再分割的个体，是否分割取决于他者的认知，而非分割的自身。
