# 129. 善与恶

生命的延续由个体和全体构成，善恶的本质也由此作为划分。利于群体的行为是善。利于个体而与群体无直接关系的是不用以善恶评价的。而由于利于个体而不利于群体的行为称之为恶。但个体又是群体的基础，所以利于个体是天性，而利于个体的同时也利于群体的是善。利于个体但不利于群体的是恶。

智能不会做不利于自身的事。做了看似不利于自身的行为只因为他人或另外时刻对于同一件事的考虑或认知方式不同。

宗教总是导人减少欲念是因为这样可减少个体因为行为对于群体的伤害。
