# 127. 拉普拉斯的兽

拉普拉斯的兽

对于系统感知的过程会对系统本身产生影响，而如果再去感知这部分，依然还会对于整个系统产生新的影响。同理，由全知去计算的过程依然是在系统内发生，而这个计算同样会对系统带来影响。而且无论感知和计算对于系统的影响都是持续的。

另外，其实固定的历史是不存在当前的，存在的只有当前状态。所以也不存在一个全知的能计算过去的兽，过去也不存在。

> 简单的说，我认为历史和未来都是确定的，但也是不可知的，任何试图了解历史或未来的方式都是整个历史的一部分，也仅此而已。
