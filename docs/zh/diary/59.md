# 59. 智能需要持续和衰减

智能的产生，意识约等于智能

1. 依赖于运动或变化，保持变化的能力或属性即惯性。否则不能构成意识的连续。

2. 依赖于物质对于变化的阻碍，即智能必须发生在一个物质范围之内。不可能出现纯能量形式的智能。因为不能变化，如一束真空中的光。

所以智能必产生于变化的变化。

意识的产生必须有「持续」和「衰减」

物质「或场」才能带来变化的变化，如果只有能量，则变化的形式不会改变。

> 宇宙作为整体也可能存在智能
