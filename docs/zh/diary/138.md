# 138. 环境短期异常

每个人的每个行为决定都是符合其主观认为符合其自身利益的。在一个理想的社会环境中，每个人行为的对错和自身利益反馈不会有冲突，所以当做决定时，表现为其行为符合普遍的对错观点。

如果在一个社会环境中，个人行为总是以利益为考量标准而非对错时，则非是个人决定时是否利于自己的天然标准改变了，而是这个社会环境的异常造成了个体利益和对错的冲突。对错既是具备时间加成的全体的利益，所以在这种情况下，可以认为利益冲突发生在长期与当下之间，也可能发生在个体与集体之间。

同时我们也应该看到这种社会环境的异常必然是短期的。
