# 194. 一切仅是在变化中

存在、消亡都是变化的过程，所以存在本身并不具备任何意义。但对于智能来说，「意义」这个评判一切的标准根基就来自于自身的存在。

物种对于个体，如同人对于细胞都可被视为尺寸维度在生命的体现。大维度的个体需要利于小维度的个体，以保持小维度个体的存在，才能有利于大维度个体自身的存在。

> 比如人需要让自己在一个适宜的温度环境下，如果温度太高，单个细胞的蛋白质便会凝固，而造成细胞的死亡，这也会影响人的存在。

单个的细胞并不能主动的选择环境，选择环境的人随是由众多的细胞构成的，但做出决定的并非全部的细胞。

但归根结底，一切仅是在变化当中而已。

> 当一个环境，比如某个规则，将所有个体的付出得到都等比例关联的时候，便已经达到了理论能够刺激个体生产的最优状态（为保护私有财产而建立国家）。比例逐渐平缓的时候，刺激也会逐渐减弱（理想社会主义）。这个比例永远不会变的平直，但是也许在某种规则下会产生巨大的扭曲，比如将付出和生产割裂成两个无关部分。此时个体争相努力付出的将会是跟生产完全无关的东西，而此规则下，所有个体的存在能力都会急剧下降（现实版1984）。
