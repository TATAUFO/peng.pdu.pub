# 204. 改变范围，缓解忧虑

正确和错误的唯一标准是是否有利于存在，所以正确和错误是相对的概念，而非绝对。同一件事，有利于个体存在的未必有利于群体存在，有利于小群体存在的未必有利于大的群体存在。所以现实中每件事情，对于不同的个体都会有不同的是否观点。但如果我们再添加一个条件，即更大的群体存在比更小的群体存在重要，如此是非观才有可能在个体间统一。

有人会因为所在群体的错误而忧虑，因为能看到这种群体的错误会导致本群体的衰弱。我们姑且暂时忽略个人行为对于群体行为的改变可能，只讨论个体如何用思维方式的改变而消除这种忧虑。一是将思维的群体范围缩小，直至缩小到自己的行为可占主导的范围，即表现为更自私，将原先自认为错误的行为变更为正确的行为。当一个群体中基于群体的错误行为多到一定规模之后，这种改变必将加速。二是将思维的群体范围扩大，当范围扩大到一定程度以后，所在的群体是否存在将不会受到我们所看到的小群体中的错误的影响，随着群体规模的增大，其相对的存在时间也会变长，长度会超过个体所能见或需要担忧的范围，就如道家思想。
