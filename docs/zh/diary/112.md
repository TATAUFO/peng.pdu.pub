# 112. 个体与层级

为什么警察、军人等要强调纪律，而在一般的企业不需要？因为警察等是因为职业而非个人的能力，经验而对民众执行强制性的管理。其本身可视为工具，通过其来体现统治者所想表达的意愿。而当警察或者城管按照自身意愿管理时，则变为无能力者执行管理。

阶层间的流动是好的，有于社会整体的，可以让底层通过自身的努力或天赋积累财富，获取权力都是好的。但如果让整个底层和上层对调则是灾难。比如宣扬贫农打到地主，再比如让物业人员管理小区。
