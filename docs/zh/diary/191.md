# 191. 理智的作用

理智只是用来看清世界，而非用来做出选择。

> 由更低的尺寸维度所带来的不确定性是根植于智能当中的，只有具备这种不确定的智能才因为其鲁棒性而能群体存在。

> 如同Common Law System即英美法系中的陪审团和律师扮演的不同角色，律师要求具有基于理智的能力，能够辨析、推理、表述清晰，而陪审团则是由一组普通公民构成，他们不需要任何专业能力，他们仅在借由律师的理智看清事情原委之后，依从自己的内心而决定是否有罪。当然，我们也可以认为最终判定是依靠陪审团成员的理智，但无专业背景要求的理智，其实已经无异于凭借感性认识而判断。
