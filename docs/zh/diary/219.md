# 219. 没代币的区块链等于数据库

并非用了区块链，记录就不可更改，没有利益的绑定，无论什么技术，都可以随便改。

比特币并非是发明了一种不可以更改的记录方式，而是发明了一种将可以用技术手段强制相对公平的分配利益方式。而且在这种方式中，最大限度的实现了当前记录者的利益和系统使用者「交易者」的利益隔离，所以使得记录者不会去有意修改记录，因为缺乏利益的驱动。

当一笔交易被记录以后，如果交易的利益相关者要改变这笔交易，则需要让利于全网所有的矿工，因为他无法预测后面的块是由谁挖出来。而没额外获利的矿工，便没有理由去挖一个可能会浪费算力的块。

从上面的过程可以看出，挖块获得奖励，下一个矿工不可预知，这套强制的利益分配方式，造成了修改记录不符合记录者自身的利益，间接导致了比特币的交易记录不能够被修改的事实。

如果改为DPoS的方式，其实已经使得这种不可修改变弱了很多，因为下一个出块矿工在某种程度上变的可预知，所以才有了所谓作恶的代表不能超过1/3的限制。但我们知道，这个1/3要比比特币的1/2容易达到的多，不单是比例问题，而是这里的1/3是可以被确定目标的1/3。换句话说就是在比特币的系统中，即是你掌握了超过1/2的算力，作恶的行为也不能保证成功。

但这些并不是关键，关键在于当区块链的技术脱离了代币以后。无论如何修改块中现存的记录，对于系统记录者利益的影响也已经脱离了这个系统，是否修改记录的原因已经变为外部原因。如同一个普通的数据库，你是否修改某项记录，取决于被发现后，这次修改这会不会被所在部门，公司或者法律制裁，而不是在这个系统中形成利益闭环。没有代币去承载利益分配，系统即便采用工作量证明，也不可能持续吸引与系统利益相匹配的算力参与。其他的共识机制也大同小异。

以记录不可篡改为目标的联盟链，无论用什么花样的算法、共识，最终都成功不了，因为其也最终也只是个数据库。区块链阻挡修改的不是记录的方式，而是利益的分配。
