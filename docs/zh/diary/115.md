# 115. 时间囚徒困境

纳什均衡所讲的囚徒困境，不止发生在不同个体之间，也可以拓展到时间维度，结果就像个体最优都不优于群体最优。每个时刻最优也非长期的整体最优。所以现状困于此，困于时间、空间的二维的囚徒困境。

囚徒间的合约可打破空间的囚徒困境。

时间的困境不易打破，除非有一个准确度较高的时时预测系统，相当于人与未来自己的合约。

「这个想法细思极恐，其破坏了智能本身制造出的未来未知。」
