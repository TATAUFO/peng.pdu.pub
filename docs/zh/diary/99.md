# 99. 零和游戏

某些社会形态因为对于社会整体物质增长的压制，迫使其中的人才由对于物质或资源的争夺，变为对于权力的争夺以间接的获取物质资源。而权力比自然资源更加稀缺，所以争夺也更加残酷。对于自然资源的争夺对于个体集整体来讲能提高效率，而权力的争夺是完全的零和游戏。

但也许马克思有一点点道理，人类最终可能进入这种社会形态，只是现在远远达不到。

> 或者说那需要的不是物质极大丰富，而是能够放弃对于物质拥有的概念。还是不可能，马克思终究不可能是对的；）起码对于人类这个物种而言。
