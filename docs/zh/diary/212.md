# 212. web3和区块链

区块链可以看成是个体意志的组织形式而web3可以认为是个体的存在形式。将其做个类比的话，区块链相当于国家，而web3相当于人类社会本身。

如果web3项目，基于区块链，那么相当于由区块链中的资产来赋予个体的身份，类似于一个人存在于国家当中，国家赋予这个人一个国民的身份。但现实却是由于信息的交流，个体构成了群体，而这个逐渐扩大链接的群体就是人类社会，在人类社会的基础之上，才有了国家。国家可以赋予一个人国民的身份的，但却不能够决定其是否为人。

构建于区块链之上的web3可以说是一个去中心化的系统，只不过这个系统是建立在某个更基础的系统之上的，而非最基础的系统。这样实现的问题是，底层区块链的运行速度必然成为这个web3系统的最大制约，无论底层的区块链运行多块，这个制约也不能消除。

> 因为社会并不需要一致性，那仅仅是货币系统的要求。

终究还是需要回到web3为最底层系统的构想。
