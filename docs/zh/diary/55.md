# 55. 单体守恒

单体守恒，加入个体集之前加入的努力等于对于加入后利益的期望。

> 单体付出收益守恒是一直持续存在的，个体间的守恒也是一直存在的。

> 单体守恒的想法并不准确，后续演化为智能的行为总是依照对于未来预期利益的最大化进行的，行为不同只是对于未来的利益预期判定的不同，有的符合事实，有的不符合事实，而且利益的判定也有动物性的和精神性的。不符合事实或者说不符合预期的情况大多数而言是对于未来事件发生预期的概率并不准确。
