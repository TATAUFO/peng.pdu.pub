# 199. 惯性和质量

惯性定律是一个伟大的定律揭示了事物的根本性质。无论能量还是物质，质量是惯性的原因或本就是一个事物的两个方面。同样在社会发展中也有明显的惯性存在。惯性会随外界的影响而改变，这种由外因引起的改变是方向的改变，而非波动本身。

> 波动是内在的性质。但究竟是外因造成的方向改变，或被归类于内因造成的波动取决于观察者的视角。

一个延续了几千年的社会形态，其惯性是无比巨大的，像一个星球，即便巨大的陨石也只能使其略微偏离方向，而要改变星球的轨道，也许本就不可能。因为足以使得星球破碎的撞击都可能小于改变轨道所需的能量。

质量是什么？质量与物维持变化状态的关系类似于有序的变化与时间的关系。也许质量也不是本质，而是物维持状态的趋势才是根本。当需要改变其变化状态时，质量才作为表现而出现。这种变化或说相对运动即是在我们认为静止的情况下依然存在，是否意味着我们存在的空间维度和我们感知的维度之间的差别，即对于相对静止的物体，也存在我们不可感知的维度下的变化。
