# 151. 解释各个时间

时间也可以认为是人所受的客观影响的结果。其实时间以人为主体。由于每个人所受的影响不同。所以每个人的时间也不同。

牛顿的时间认为在空无一物处有时间。这只是每个观点对时间认识的不同。他认为时间是绝对的。或牛顿认为的时间就是光在场中的波动周期为基础。而相对论时间是以光为标准的感受为准，而计算的。其已经开始通过更精确的实验方式测出不同主体的时间差别。为了适用于平衡设备由光或者电的速度带来的影响。所以将时间定义变为至于要测的光相关的定义。

由于只有被测物体能接近光速。所以也适用于当下。但局限在于其忽视了当自身高速时，各方向的感受影响不均，时间失效。

> 自身高速特指一个观测者以相对光速在某环境中运行的情况。好比坐在光速列车上看这个世界。
