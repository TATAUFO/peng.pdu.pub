# 190. 扳道岔的正义

「正义」的第一课上主要在讨论，一辆无法停止的火车驶来，如果你什么都不做，火车会撞死前面铁轨上的5个人，而如果搬动道岔，火车则会撞死另一条铁轨上的1个人，你会如此做？或者说何种做法是正义？

当真正接受存在就是存在本身唯一的意义，则这个问题便容易解释。

如果这件事的影响范围仅限于上述几个人，则搬动道岔无疑是正义的行为，因为这个绝对理智的行动最大程度维护了由着几人构成的群体的存在。但如果这个世界并非只有这几个人，事件也会扩散出去被大众知晓，则这种绝对理智的逻辑导向会破坏社会本身的多样性，或去中心化。因为人们越理智，其行为选择越单一。这样的结果是弱化了社会作为系统的鲁棒性，使得系统更加脆弱，变得不利于群体的存在。

当面临抉择的时候听从自己内心的呼喊，其实是在维持群体的多样性。当面临抉择，必然是在面对失去和得到相对平衡的情况，此时跟随自己的内心其目的并不是为了自己，而是为了全体。

当面临简单的情况，人需要理智而非情感。当面临复杂和未知，比如搬动道岔的时候，并不知未来需要面对社会怎样的对待，赞扬或是谴责，甚至因为你的行为造成死亡的无辜者亲人的报复时，跟随自己的内心，做个随机的决定就好，这是降低系统性风险的最好的选择。

面对任何情况，需要用理智剥离无关的事项，如果利弊明显就选有利的，如果依然太复杂，有太多的未知无法判断，则随便选一个就好。此时并非你的选择是正义，而是那种跟随内心的随机选择方式，已经就是正义了。

> 洞穴奇案那本书所讨论的内容也类似。
