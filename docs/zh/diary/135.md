# 135. 时间失效

时间在微观量子状态下会失效。因为尺寸维度影响下被视为物质个体过少，因而发生的事件也少。所以不能构成有序的统计结果，只有有序的事件。

> 测不准，因为观察本身的影响已经不可忽略

时间在高速，如何光速相比不可忽略时也会失效。因为个方向的统计结果不一致。比如相对于以太「这里的以太只是借用一个名字，并非牛顿的以太」，即空间场高速运动时，如果依然按照原来的时间的概念理解，那前方感受时间变快，后方时间变慢。左右不变。这是一种无法想象的怪异感受。
