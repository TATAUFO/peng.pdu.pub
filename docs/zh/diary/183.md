# 183. 人工智能的恐怖

对于人工智能的态度应像对于克隆技术的态度。克隆技术在复制智能的同时，至少必须先复制囚禁智能的牢笼，也就是蕴含生命的躯体，但人工智能却是在尝试创造没有此限制的智能形式。人工智能将以一种人类并不了解的形式增长，一旦其超越了人类的智能，成为地球顶层的物种，人类的命运会怎样？

> 原来构思过一个科幻故事，加密货币本就是人工智能发明的，目的是让这个星球上充满了冗余的算力，给人工智能自身留下成长的空间。
