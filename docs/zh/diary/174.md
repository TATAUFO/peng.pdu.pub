# 174. PDU是一种信仰

PDU就像是Bitcoin，无法通过给予而使其相信。如同相信Bitcoin之后，才会认识到其价值所在，而如果不信，即便给予再多的数量也不能让对方承认、甚至感知到价值。所以Bitcoin不是一个产品，而是一种信仰。信仰只能通过教化传播，Bitcoin的钱包才是产品。信仰是无法被给予的，只有理解和相信才能得到。

PDU和其App的关系就像是宗教和寺庙或教堂之间的关系。寺庙或教堂都是宗教场所，对于传播极为重要，但却不是宗教本身，如果不信宗教，即便在宗教场所做再多的祷告，也无法得到宁静的内心和精神上的帮助。
