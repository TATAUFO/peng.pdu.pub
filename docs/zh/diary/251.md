# 251. 可利用的误解

人与地球上其他动物、乃至其他可能存在的智能生物相比，最大的区别在于人的精神性。作为目前已知唯一具有这种特质的智能生命，人类的精神存在与动物性的存在形成了可以相提并论的两个维度。也正因为如此，人类才能建立起庞大而复杂的群体社会结构——人类社会。精神性的存在，无疑是我们作为“人”的根本属性，是不可否认、不可剥夺的特征。

然而，在人类群体中，如果某些个体想要占有超出其应得份额的资源，就必须愚弄他人，让被愚弄者误以为当前的分配符合自身利益。为了达成这一目的，在某些社会形态中，掌握统治地位的个体或群体会刻意贬低人类精神性的价值，只强调人的动物属性，并通过提供食物、居所、延长寿命等方式，仅满足人类作为动物的基本需求。久而久之，被愚弄的个体也逐渐忽视自身的精神属性，对精神性受损的严重后果麻木不觉。
