# 173. 信息公开的基础

信息的获取并不代表一个身份，或者说并不影响一个个体的定义。只有信息的表达，才能参与定义一个身份，因为这种行为才会在个体集中留下个体的痕迹。所以信息的获取过程本不需要身份的介入，这也是为何在PDU系统中信息全部公开的基础。

对于还未完全接受PDU想法的人来说，理解这件事是相对困难的。举个例子，在路边一个没有任何监控的地方放着一锭黄金，如果没有人知道那么黄金不会被拿走。但是如果有人在散步的时候看到的这锭金子，就有很大的可能性捡走这锭黄金。不接受PDU的人会认为是获知金子位置信息引发了后续的行为，但我们认为如果没有捡起黄金的行为，前面的信息获知过程并不会对世界有任何影响。

或者从另一个角度去看，我们只关注于整个系统，而不关注在个体内部所发生的变化。
